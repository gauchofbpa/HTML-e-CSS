Bem-vindo ao site explicativo, aqui estará uma introdução a Tags do HTML

Por: João Constantino Caetano - DS24/01 M2

html

Começando pela tag básica que você irá precisar usar em qualquer site que for criar, a tag html é como se representasse a raiz do documento, ela engloba todos os outros elementos do site

head

A tag head é utilizada para o navegador interpretar algumas informações, como por exemplo o título do site, especificar o conjunto de caracteres e etc.

body

Na tag body, é onde fica todo o conteúdo de texto, imagem e vídeos, em que o usuário que acessa o site vê e interage.

header

A tag header serve para indicar para os navegadores onde está o cabeçalho do site.

section

A tag section é usada para marcar seções de conteúdo de uma página.

div

Essa tag pode ser utilizada para agrupar ou separar conteúdos.

footer

Na tag footer, ela marca a área inferior (o rodapé) do conteúdo geral, ou do conteúdo de uma seção.

h1, h2, h3, h4, h5 e h6

Essas tags servem para dar um destaque no texto, usando para título ou subtítulos. Elas variam de tamanho, aqui está um exemplo de cada uma:

h7

Depois do h6, o texto diminuí tanto de tamanho que o site já não aceita mais, ele fica como um texto normal. Então, o máximo é h6!

hr

Essa tag cria uma linha no site, podemos utilizar para separar visualmente o site. Vale ressaltar que é um elemento vazio, que não precisa de tag de fechamento.

p

A tag p é utilizada para criar um parágrafo de texto.

ul

Nessa tag podemos criar uma lista não ordenada, com marcadores e não números. Segue o exemplo:

  • Batata doce
  • Alface
  • Tomate

li

A tag li é usada para representar um item que faz parte de alguma lista, então obrigatoriamente essa tag deve estar contida em uma lista, como por exemplo a não ordenada (ul).

br

Essa tag é para adicionar quebras de linha em um texto.

strong

Essa tag é utilizada para dar um destaque de destaque, exemplo: O Grêmio foi roubado.

em

A tag da um destaque em itálico, exemplo: O campo está ruim.

u

Essa tag sublinha o texto, exemplo: Amanhã choverá.

ol

Nessa tag, criamos uma lista ordenada dessa vez, com números. Segue o exemplo:

  1. Arroz
  2. Feijão
  3. Carne

nav

A tag nav define um conteúdo de navegação, utilizado em conjunto com listas e criação de menus.

aside

Essa tag representa uma seção de uma página em que o conteúdo é relacionado ao conteúdo do seu entorno, mas que pode ser separado do conteúdo.

article

Nessa tag é definido um artigo para a página, separando o conteúdo da sua página, muito utilizado por blogs.

img

A tag img serve para incluir uma imagem ao texto.

video

Essa tag é usada para fazer a inserção de um vídeo.

audio

Nessa tag podemos inserir um aúdio a página, funciona da mesma forma que o de vídeo.
